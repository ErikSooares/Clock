O projeto Clock é um pequeno projeto web que consiste em duas abas diferentes em uma página. A primeira aba exibe as horas atuais, incluindo horas, minutos e segundos. Esses dados são capturados pelo JavaScript a partir do relógio do computador do usuário, proporcionando uma exibição em tempo real.

Já a segunda aba exibe um cronômetro que inicialmente está zerado. Ao clicar no botão "Iniciar", o cronômetro começa a contar e o botão muda seu rótulo para "Zerar". Isso permite que o usuário inicie, pare e, se desejado, zere o cronômetro.

Resumidamente, a funcionalidade do projeto é fornecer um display das horas atuais na primeira aba, enquanto na segunda aba, oferece um cronômetro com as opções de iniciar, parar e zerar. É uma aplicação simples, mas útil para exibir a hora atual e para utilizar um cronômetro básico.

## Funcionalidades

Aba 1: Horas Atuais
Exibe as horas atuais (horas, minutos e segundos) capturadas do relógio do computador do usuário.

Aba 2: Cronômetro
Iniciar: Inicia o cronômetro.
Parar: Pausa o cronômetro.
Zerar: Reinicia o cronômetro para zero.

## Uso
Abra o arquivo HTML em um navegador web

Na primeira aba, as horas atuais serão exibidas automaticamente.

Na segunda aba, o cronômetro começa zerado.

Clique no botão "Iniciar" para começar a contagem do cronômetro.
Use os botões "Parar" e "Zerar" conforme necessário.
